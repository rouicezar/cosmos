# 小学数学可视化站点开发说明

## 技术选择

第一版使用静态 HTML、CSS、JavaScript：

- 无构建步骤。
- 无第三方依赖。
- 可直接打开 `index.html`。
- 测试使用 Node.js 读取源码做数据完整性与结构校验。

## 文件结构

- `index.html`：应用入口。
- `styles.css`：视觉样式与响应式布局。
- `app.js`：知识点数据、状态管理、实验渲染器。
- `scripts/test.js`：本地测试脚本。
- `docs/requirements.md`：需求文档。
- `docs/design.md`：设计文档。
- `docs/development.md`：开发文档。

## 实现阶段

1. 建立静态应用骨架。
2. 整理 1-5 年级知识点数据。
3. 实现年级、搜索、知识点切换。
4. 实现实验渲染器与交互控件。
5. 增加响应式和视觉细节。
6. 编写并运行测试。

## 测试策略

`scripts/test.js` 检查：

- 关键文件存在。
- 1-5 年级都有数据。
- 每个知识点都有必要字段。
- 每个知识点的 `visual` 都有对应渲染器。
- 页面入口引用了样式和脚本。

## 提交与推送

仓库当前为空且没有 `.git`。实现与测试通过后，可以初始化 git 并提交。推送需要远程仓库地址；如果没有远程，只能完成到本地提交阶段。
